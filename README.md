# Nassau Candy Shipping Efficiency Analysis

This project analyzes factory-to-customer shipping efficiency using the Nassau Candy Distributor dataset.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## Key Analysis
- Shipping efficiency by region
- Delivery delay detection
- Product sales analysis
- Route efficiency analysis

## Objective
Identify logistics inefficiencies and provide recommendations to improve delivery performance.
